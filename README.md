# thisiskinglis.com — GitHub Pages site

This is a simple, editable static site for KINGLIS.

## Structure (edit these files)
- `index.html` — homepage sections: This Is, Believe, Think, Do, Who, Difference, About, Contact
- `assets/css/style.css` — colors, fonts, spacing
- `assets/img/logo.png` — your logo
- `resources/` — drop PDFs, decks, case studies here
  - Edit `resources/index.html` to add a link block for each file

## How to deploy to GitHub Pages
1. Create a new public repo named `thisiskinglis` (or `kaylee-jane.github.io`)
2. Upload all files from this folder to the repo root
3. In GitHub: Settings → Pages → Deploy from main branch / root
4. Add custom domain: `thisiskinglis.com`
5. Add a `CNAME` file (already included) with `thisiskinglis.com`
6. In your DNS (where you bought the domain): add the GitHub Pages A records

## Editing content
- Open `index.html` and edit text between the section comments
- To add a resource: upload PDF to `/resources/`, then duplicate a `.resource-item` block in `resources/index.html`
